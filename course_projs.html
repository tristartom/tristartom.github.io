CS 6238 Computer System Security.
  1. Course project: Tolerate password case mispelling wo/ losing utility, [coding_logs], [src available on request].
    Description: The project aims at building a toolkit that enable users to customize his/her preferences on password case sensitivity, while without losing security. For instance, the user choosing password "tang" may want the case-sensitivity to be less rigid; say "Tang", "taNg" and "tanG" are all acceptable password. On implementation, the project essentially adopts a secure authentication scheme [link] by choosing case of password as feature vectors. The scheme uses secret sharing to "hash" features of character cases, with secret being a key for encryption. For implementation details, pls refer to my src and coding log, as linked above.
