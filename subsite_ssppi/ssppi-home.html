Privacy Preserving Indexing for eHealth Information Networks


 


Welcome to SS-PPI Demo Web Site

SS-PPI is a a novel indexing scheme that supports large-scale distributed information network comprising thousands of content providers, and provides theoretical guarantees on both possession privacy protection and execution efficiency

The demo program is generated as Java classes. It includes several modules such as building providers network map based on GeoGrid, privacy group construction, group index construction, global index construction, PPI build, term query and provider addition. Compared with existing privacy-preserving index schemes, our solution highlights with a series of distinct features:

  • access control policies incoporation - It incorporates access control policies in the privacy-preserving index, which improves both search efficiency and attack resilience.
  • fast index construction - It employs a fast index construction protocol via a novel use of the secrete-sharing scheme in a fully distributed manner (without trusted third party), requiring only constant (typically two) round of communication.
  • information-theoretic security assurance - It provides information-theoretic security against colluding adversaries during index construction as well as query answering.
