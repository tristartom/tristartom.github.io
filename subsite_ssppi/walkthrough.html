Walkthrough
( Phase 1 | Phase 2 | Phase 3 | PPI Maintainance )


 


Phase 1 - Initialization and Group Construction

  1. After launching the demo application, there are 3 options in the first drop-down menu. Make sure you select "SS-PPI" as this is the only supported scheme in the current demo.

  2. Click on the 1-click PPI Build button if you want to see the final PPI index result in one click. The demo will pop-up two windows as follows: the left one is the list of providers and the right one is the final PPI index which will be published to the index server.

  3. By default there are 100 providers in the network. You can enter details for a specific provider by clicking on Enter Provider Info button after selecting the wanted provider in the drop-down menu above

  4. Click on the Build Network button to randomly place providers onto the network map. The demo will create a map in the right panel as the following picture:

  5. There are 3 options for Group Construction. Currently only Random grouping is supported. After selecting Random in the drop-down menu, click on the Group Construction button. All the participating providers will be organized into disjoint privacy groups. Each group has different color in the network map. The demo also pop-up a window containg the list of providers of each group.

  6. You can also choose to change the default group color by clicking on the Set Group Color button after choosing the group you want to change color in the drop-down list above.


Phase 2: Build Local Group Index

  1. Click on the Build Group Index button to build index structure for all groups. For each group, all the terms belong within this group will be displayed , each together with the number of group providers having the term.

  2. Now if you want to see the index of a specific group, you can do that by clicking on the Display Group Index button after selecting the group you want to display in the appropriate drop-down menu.

Phase 3: PPI Build & Term Query

  1. Click on the Global Index button to build global index. Each term possessed by providers in the network will be displayed together with the list of groups having this term.

  2. Click on the PPI Build button to build the final index structure. Each term possessed by providers in the network will be displayed together with the list of providers actually having this term.

    .

  3. Click on the Term Query button to input the term you want to query and search for providers who possess this term. In the following, the left window will be pop-uped to enter the term. After clicking OK button, the returned providers will be showed as in the right screenshot.

PPI Maintainance

  1. You may add more providers one by one to the system by clicking on the Add Provider button. The Enter Provider Info window will be displayed and after entering details for new provider, the following confirm message will be popup.

  2. Click on the Update Index button to rebuild the index structure after the new provider has joined.
