Project (CIS 628)

Timeline

  • Topic proposal: Oct. 04, 2019
  • Final submission: Dec. 08, 2019

Deliverables

  • Proposal (15%): one-pager including abstract.
  • Final report (85%): Formal survey of the technology with no less than 10 pages (format: 12-point font size on single-spaced 8.5" x 11" pages). The report should focus on a cryptography-centric topic and needs to have the following elements for each technique surveyed: scheme definition, security definition, sample constructions, security analysis, proof and/or intuition.

Sample topics

  • Homomorphic encryption
  • Multi-party computations
    • Garbled circuits, oblivious transfer, etc.
  • Verifiable computations
  • Zero-knowledge proof
  • Adversarial machine learning and adverserial examples
  • Public blockchains
